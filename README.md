# Blinkit-Sale-Dashboard
This project presents an interactive Power BI dashboard built to analyze and visualize key business metrics for Blinkit, an online grocery delivery platform. The dashboard provides actionable insights into sales performance, customer behavior, product categories, and outlet-level analysis to support data-driven decision-making.

Key Features
Interactive and user-friendly Power BI dashboard
Sales and revenue analysis
Category-wise and product-wise performance insights
Outlet and location-based analysis
Trend analysis using dynamic filters and slicers
KPI cards for quick business overview

Tools & Technologies
Microsoft Power BI
Data Modeling & DAX
Data Visualization

Use Case
This dashboard is useful for: Business performance tracking
Identifying top-performing products and outlets
Understanding customer demand patterns
Supporting strategic and operational decisions
